Soft‑AP (fallback / AP-only mode)
📶 What happens after restart?

The device will reboot and apply Wi‑Fi mode, STA credentials, and soft‑AP overrides where configured.

STA-first mode tries your saved network first; if it fails, the soft‑AP starts using your custom AP SSID/password when set (otherwise the factory defaults).

✅ If connection succeeds: the assigned IP address will be printed in the serial console, and shown on the display (if available).
⚠️ If connection fails: the device will automatically fall back to AP mode, so you can reconnect and reconfigure.
